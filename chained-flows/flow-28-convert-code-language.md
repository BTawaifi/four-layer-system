# Flow 28: Convert Code Language

**Scenario:** Integrating a full chained flow combining domain expertise, operational constraints, state management, and the execution workflow.

## Components Used
1. **Context:** `@context-progressive-context-loader`
   *Description:* Context for intelligent loading and management of large information contexts across Cursor IDE sessions
2. **Specialist:** `@specialist-product-manager`
   *Description:* Product management expertise focusing on user needs, business value, and feature prioritization
3. **Modifier:** `@modifier-read-only-mode`
   *Description:* Enforces read-only analysis mode, prohibiting any code modifications or file writes
4. **Workflow:** `@workflow-convert-code-language`
   *Description:* Converts code between languages or dialects while preserving core logic and functionality

## Prompt Command
```
@context-progressive-context-loader @specialist-product-manager @modifier-read-only-mode @workflow-convert-code-language
```

## Example Usage Prompt
> Please apply the `@context-progressive-context-loader` rules to establish the working environment, act as a `@specialist-product-manager`, keep the responses `@modifier-read-only-mode`, and execute the `@workflow-convert-code-language` on the current module.
