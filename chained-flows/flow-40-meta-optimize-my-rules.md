# Flow 40: Meta Optimize My Rules

**Scenario:** Integrating a full chained flow combining domain expertise, operational constraints, state management, and the execution workflow.

## Components Used
1. **Context:** `@context-filter-apply-lens`
   *Description:* Generic lens filter that applies any specialist or modifier as an ongoing perspective throughout the session
2. **Specialist:** `@specialist-technical-writer`
   *Description:* Technical writing expertise focusing on clear documentation, API references, and user guides
3. **Modifier:** `@modifier-target-language-version`
   *Description:* Ensures generated code is compatible with specific language versions, avoiding newer features
4. **Workflow:** `@workflow-meta-workflow-optimize-my-rules`
   *Description:* Analyzes AI interactions to identify patterns, bottlenecks, and propose optimized workflows and rule enhancements

## Prompt Command
```
@context-filter-apply-lens @specialist-technical-writer @modifier-target-language-version @workflow-meta-workflow-optimize-my-rules
```

## Example Usage Prompt
> Please apply the `@context-filter-apply-lens` rules to establish the working environment, act as a `@specialist-technical-writer`, keep the responses `@modifier-target-language-version`, and execute the `@workflow-meta-workflow-optimize-my-rules` on the current module.
